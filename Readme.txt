Readme
======

Given the following arrays:

            string[] array1 = { "ABC", "DEF", "GHI" };
            string[] array2 = { "ABC", "KLM", "NOP" };
            string[] array3 = { "ABC", "QRS", "STU" };
            string[] array4 = { "ABC", "VXY", "Z12", "345" };
            
 Concatenate given arrays assuming that the first element in each array is the header which should appear only once at the top of the concatenated array:
 
 ie.
 
ABC
DEF
GHI
KLM
NOP
QRS
STU
VXY
Z12
345
